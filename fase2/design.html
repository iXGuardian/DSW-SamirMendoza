Propósito del sitio

En este sitio, vamos a realizar el estudio de las etiquetas basicas de HTML asi como la personalizacion de sus atributos valores, esto tambien con el fin de poner en practica los temas de la fase 2 del curso de diseño de sitios web

Para escribir codigo HTML solo necesitamos un editor de texto sencillo, para el cual recomendaremos SublimeText o Visual Studio Code, nos introduciremos en el mundo del Html y todos sus principios basicos, para ello utilizaremos de referencia el libro HTML5, CSS3 y JQuery: curso práctico de Recio García, J.
