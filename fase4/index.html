Propósito del sitio

En este sitio, vamos a realizar el estudio de las etiquetas basicas de HTML asi como la personalizacion de sus atributos y valores, de esta manera podremos crear una base solida para nuestra pagina web

Introducción

Para escribir codigo HTML solo necesitamos un editor de texto sencillo, para el cual recomendaremos SublimeText o Visual Studio Code, nos introduciremos en el mundo del Html, CSS y todos sus principios basicos, para ello utilizaremos de referencia el libro HTML5, CSS3 y JQuery: curso práctico de Recio García, J.
